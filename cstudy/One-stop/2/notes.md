# 第2章 常量、变量和表达式

## C标准规定的转义字符

| 符号 | 含义                  |
| ---- | --------------------- |
| `\'` | 单引号 '              |
| `\"` | 双引号 "              |
| `\\` | 反斜线 \              |
| `\?` | 问号 ?                |
| `\a` | 响铃（Alert或者Bell） |
| `\b` | 退格（Backspace）     |
| `\f` | 分页符                |
| `\n` | 换行                  |
| `\r` | 回车                  |
| `\t` | 水平制表符            |
| `\v` | 垂直制表符            |
